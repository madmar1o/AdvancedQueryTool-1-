Невозможно подключиться к Информационной базе!

Проверьте, что:

  1. антивирус или брандмауэр не блокируют запуск компонентов Агента.
  2. в базе не выполняются длительные регламентные задания, которые устанавливают блокировку на файл ИБ;

Попробуйте перезагрузить компьютер с Агентом, временно отключите антивирус и брандмауэр, переустановите Агент согласно руководству пользователя.

Если ничего не помогает и вы все равно видите это сообщение, пришлите нам на адрес технической поддержки support@1c.link следующее:

  1. файл agent-log.txt;
  2. тип публикуемых информационных баз (файловый, клиент-серверный);
  3. диагностическую информацию программы “Сведения о системе” о ПК с Агентом.
